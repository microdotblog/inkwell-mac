# Inkwell for Mac

If you use the code in another project, please don't use the name Inkwell or Micro.blog unless it is obviously distinct from the product name. For example: "Your App for Inkwell" is fine, but "Inkwell 2.0" is not. Also don't use the icon unless it is to refer directly to the Inkwell syncing service. The code is open source, but the icon is not.
